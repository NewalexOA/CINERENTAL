# CSS Libraries

Эта директория содержит сторонние CSS библиотеки, необходимые для работы приложения:

* `daterangepicker.min.css` - Стили для компонента выбора диапазона дат

## Внимание

Эти файлы должны быть включены в Docker-образ при сборке. Они должны быть доступны по пути:

- `/app/frontend/static/css/lib/daterangepicker.min.css`

При изменении Docker-образа убедитесь, что эти файлы включены.
